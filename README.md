# Expense Tracker

A simple Expense Tracker web app built using HTML, CSS, and JavaScript.

## Features
- Add income and expenses
- Delete transactions
- Calculate balance
- Track spending
- Store data using Local Storage

## Tech Stack
- HTML
- CSS
- JavaScript

## Learning Goals
This project is being built to practice:
- DOM Manipulation
- Events
- Arrays & Objects
- Local Storage
- Functions
- Form Handling
